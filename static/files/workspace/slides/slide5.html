TRANSFORMER INTERNALS

NORMALIZATION

Post-norm -> Pre-norm -> RMSNorm

Post-Norm (2017)

LayerNorm after residual: harder gradient flow in deep stacks.

LayerNorm(x + F(x))

Pre-Norm (Modern)

Residual is a clean highway; stable training at depth.

x + F(LayerNorm(x))

RMSNorm + QK-Norm

Drop mean subtraction. Faster, fewer params, stable.

RMSNorm(x) = x / RMS(x)
