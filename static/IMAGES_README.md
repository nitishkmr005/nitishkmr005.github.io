# Images Directory

This directory contains images and assets for your portfolio website.

## Required Images

### Profile Picture
- **File**: `profile.jpg` or `profile.png`
- **Size**: 400x400px (minimum)
- **Format**: JPG or PNG
- **Usage**: Displayed on the home page

### Favicons

Generate favicons using [Favicon Generator](https://realfavicongenerator.net/)

Required files:
- `favicon.ico` (16x16, 32x32, 48x48)
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png` (180x180)

### Optional Images

- **Blog post covers**: Store in `static/images/posts/`
- **Project screenshots**: Store in `static/images/projects/`
- **General images**: Store in `static/images/`

## Image Guidelines

1. **Optimize images** before uploading
   - Use tools like [TinyPNG](https://tinypng.com/) or [ImageOptim](https://imageoptim.com/)
   - Target file size: < 500KB for profile pic, < 200KB for blog images

2. **Naming convention**
   - Use lowercase
   - Use hyphens instead of spaces
   - Be descriptive: `ml-model-architecture.png` instead of `image1.png`

3. **Formats**
   - Photos: JPG (better compression)
   - Graphics/logos: PNG (transparency support)
   - Icons: SVG (scalable)

## Quick Setup

1. Add your profile picture as `static/profile.jpg`
2. Generate and add favicons to `static/`
3. (Optional) Add blog post images to `static/images/posts/`

---

*All images should be your own or have proper licensing/attribution.*
